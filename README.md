# Banner Studio

A GitHub-ready Next.js banner mockup application with automatic GitHub Pages deployment.

## Publish with GitHub Pages

1. Create a new public GitHub repository, such as `banner-studio`.
2. Upload every file and folder from this package to the repository root.
3. Open **Settings → Pages** in the GitHub repository.
4. Under **Build and deployment**, set **Source** to **GitHub Actions**.
5. Open the **Actions** tab and wait for the “Deploy Banner Studio to GitHub Pages” workflow to finish.
6. GitHub will provide the published URL in **Settings → Pages**.

The workflow automatically detects the repository name and configures the correct URL path.

## Local development

```bash
npm install
npm run dev
```
